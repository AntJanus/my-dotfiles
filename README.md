## Ant's Dotfiles

To properly make this all work, you need to install a bunch of tools!

1. Node + NVM
2. zsh and oh-my-zsh
3. neovim
4. rofi (dmenu replacement)
5. lsd (ls substitute)
6. bat (cat substitute)
7. slapmoji (emojis)
8. ranger (file manager)
9. conky


NPM packages to install:

```sh
npm install -g \
	commitizen \
	cz-conventional-changelog
```
